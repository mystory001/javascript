배열 합치기(배열 연결)

배열명.concat(합칠(연결할) 배열명) 메소드

배열 내 요소끼리 합치기

join('구분자지정') 배열 요소 사이에 지정한 구분자를 넣어서 문자로 만듦
배열내의 각 요소들을 연결해서 하나의 문자열로 만듦

새로운 요소 추가 : length도 같이 반환함

배열 맨 끝에 요소 추가 : 배열명.push(요소값)
배열 처음에 요소 추가 : 배열명.unshift(요소값)

요소값 제거

배열 맨 끝에 요소 제거 : 배열명.pop(요소값)
배열 처음에 요소 제거 : 배열명.shift(요소값)

원하는 위치에 요소를 추가, 삭제 splice()

원본에 변경이 일어남
한 번에 요소를 2개 이상 추가하거나 제거(새로운 배열로 결과값을 반환)

기존 배열을 바꾸지 않으면서 요소를 꺼내는 slice()

원본 배열에 변경이 없음
